# 위핀(wepin)의 공개 코드 입니다.

장소 공유 소셜 네트워크 서비스 **위핀(wepin)** 입니다.

## 기능

1.  회원가입
    -   유효성 검사
        -   영문, 숫자, 밑줄 및 마침표를 포함한 6자 이상 입력
2.  게시글
    -   게시글 리스트
    -   게시글 작성
    -   게시글 삭제
3. 댓글
    -   댓글 작성
    -   댓글 삭제
4. 상태관리
    -  Redux
5. 기타
    -  utils 함수
    -  PrivateRoutes
    -  Style (SASS, Styled-components)
